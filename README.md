## Directory Descriptions

Codes/
Contains Jupyter notebooks implementing five cumulative regression test selection algorithms: Aging, Deficit Round Robin (DRR), Memory-less, Round Robin (RR), and Weighted Round Robin (WRR).

Data/
Contains the two datasets used in all experiments: D1.xlsx and D2.xlsx.

Results/
Stores the result files generated by running each algorithm on both datasets: Results_D1.xlsx and Results_D2.xlsx.

Workflow: How to Run the Experiments
Just run the pipeline with either one of the dataset and it will automatically generate. Run the first block it'll install all dependencies --> ask for i/p (upload the D1/D2) --> results will be generated
